#  Abady E-Commerce Website UI

**Abady E-Commerce Website UI** is a responsive multi-page e-commerce website user interface designed and developed using **HTML & CSS**.  
The project focuses on clean structure, modern design principles, and a user-friendly shopping experience across different devices.

---

##  Key Features
- Responsive design optimized for different screen sizes
- Clean and modern UI layout
- Well-structured product sections and categories
- User-friendly navigation and interface
- Organized and reusable CSS styling

---

##  Multi-Page Layout
- Home Page  
- About Page  
- Shop Page  
- Product Page  
- Shopping Cart Page  
- Contact Page  

---

##  Tech Stack
- HTML5  
- CSS3
